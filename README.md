# Devjobs Web App

## Overview

This project is a jobs board web application built using HTML, CSS, and JavaScript. The app loads job data from a local `data.json` file and displays it dynamically.

The goal is to build a clean, responsive interface that matches the provided design as closely as possible.

---

## Features

Users should be able to:

- View an optimized layout for different screen sizes
- See hover states for interactive elements
- Filter jobs by title, location, and full-time status
- Click on a job to view detailed information
- (Bonus) Automatically apply light or dark theme based on system preferences using `prefers-color-scheme`

---

## Project Structure

- `/assets` – Images and static assets
- `data.json` – Local job data source
- HTML files – Page structure
- CSS files – Styling and responsive design
- JavaScript files – Data loading and interactivity

---

## Development Guidelines

Suggested workflow:

1. Review the design and plan your layout.
2. Structure the HTML first.
3. Add base styles (typography, layout, spacing).
4. Build responsive layouts using media queries.
5. Implement filtering and dynamic job rendering with JavaScript.
6. Add theme support (optional bonus).

---

## Deployment

You can deploy this project using:

- GitHub Pages
- Vercel
- Netlify

Make sure your `index.html` is in the correct root folder before deploying.
